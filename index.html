1. Document Structure

Minimal Document

Every LaTeX document starts with a document class and wraps content in the document environment.

\documentclass{article}
\begin{document}
Hello, World!
\end{document}

Preamble Anatomy

The preamble is everything before \begin{document}. It contains package imports, custom commands, and document settings.

\documentclass[12pt,letterpaper]{article}
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage{amsmath,amssymb,amsthm}
\usepackage{graphicx}
\usepackage{hyperref}

\title{My Document}
\author{Author Name}
\date{\today}

\begin{document}
\maketitle
% Content here
\end{document}
Document Classes

article

For short documents, scientific papers, journal articles.

\documentclass[11pt,a4paper,twocolumn]{article}

report

For longer documents with chapters, theses, books.

\documentclass[12pt,oneside]{report}

book

For full books with front/main/back matter.

\documentclass[twoside,openright]{book}

memoir

Enhanced book class with more layout options.

\documentclass[14pt,extrafontsizes]{memoir}

beamer

For presentations and slides.

\documentclass[aspectratio=169]{beamer}

IEEEtran

For IEEE journal and conference papers.

\documentclass[conference]{IEEEtran}
